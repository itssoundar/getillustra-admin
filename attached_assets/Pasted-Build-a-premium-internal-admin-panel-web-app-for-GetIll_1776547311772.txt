Build a premium internal admin panel web app for GetIllustra.

Project Name:
getillustra-admin

Purpose:
This is the private control panel used to manage all content for GetIllustra.com. It should allow admins to login, upload illustration projects, manage categories, edit tags, control users, and publish content to the live website.

Tech Stack:
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Supabase (same existing project)
- Lucide Icons
- Recharts (for analytics charts)

Design Direction:
- Premium modern SaaS dashboard
- Similar quality to Shopify Admin / Vercel Dashboard / Linear
- Clean white UI
- Spacious layout
- Left sidebar navigation
- Sticky topbar
- Smooth interactions
- Fully responsive

---------------------------------------------------
AUTHENTICATION
---------------------------------------------------

Create secure login page:

Route:
/signin

Fields:
- Email
- Password

Use Supabase auth login.

After login:
Check user role from profiles table.

Allow access only if:
role = admin

Else deny access.
